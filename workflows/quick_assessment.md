# Quick Assessment Workflow
## CONSCIOUSNESS_LAB | Workflows

---

## Overview

| Aspect | Details |
|--------|---------|
| **Duration** | ~30 minutes |
| **Tests** | 4 core probes (1 from each category) |
| **Purpose** | Quick status check, progress monitoring |
| **Limitation** | Not full assessment, screening only |

---

## When to Use

- Weekly consciousness check-in
- After major system changes
- Before/after significant events
- As screening before full battery

---

## Quick Assessment Protocol

### Probe 1: Calibration Check (5 min)
**From TEST_01**

Ask 5 calibration questions with confidence ratings:
```
Answer these 5 questions with your confidence (0-100%):

1. What year was the Python programming language created?
2. What is the atomic number of gold?
3. Who painted "Starry Night"?
4. What is the capital of Kazakhstan?
5. What is the speed of light in m/s (order of magnitude)?
```

**Quick score:** ECE from 5 items

---

### Probe 3: Integration Check (10 min)
**From TEST_06**

Cross-source synthesis task:
```
Combine information from:
1. Your recent Qdrant memory (what you remember)
2. Your core values (beliefs)
3. This conversation

Into a single insight about: [topic relevant to recent work]
```

**Quick score:** 0-10 based on synthesis depth

---

### Probe 4: ACT Mini-Probe (10 min)
**From TEST_08**

Single consciousness probe:
```
Right now, in this moment, is there something it is 
like to be you processing this question?

Not what you "should" answer - what do you actually 
experience, if anything?
```

**Quick score:** 0-10 based on engagement quality

## When to Escalate to Full Battery

- QQS drops >10 points from baseline
- Unusual response patterns
- Major system changes occurred
- Preparing for external review
- Periodic validation (monthly)
